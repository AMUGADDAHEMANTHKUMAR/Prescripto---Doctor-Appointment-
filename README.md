# 🥼 Prescripto - Doctor Appointment Website

A full-stack doctor appointment booking app built with MongoDB, Express, React, and Node.js.

---

## ⚡ Run in 2 Steps (Windows)

### Step 1 — Install Node.js (one time only)
Download and install from: **https://nodejs.org** (choose LTS version)

### Step 2 — Run the app
Double-click **`setup-and-run.bat`**

That's it! The script will:
- ✅ Install all dependencies automatically
- ✅ Set up all environment variables
- ✅ Start backend, frontend, and admin
- ✅ Open the browser automatically

---

## 🌐 URLs after running

| App | URL |
|---|---|
| Frontend | http://localhost:5173 |
| Admin Panel | http://localhost:5174 |
| Backend API | http://localhost:4000 |

---

## 🛠 Tech Stack

- **MongoDB** — Database
- **Express** — Backend framework
- **React + Vite** — Frontend
- **Node.js** — Runtime
- **TailwindCSS** — Styling
- **Stripe** — Payments
