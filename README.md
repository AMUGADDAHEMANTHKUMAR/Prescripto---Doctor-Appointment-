# 🧠 Prescripto AI - Smart Doctor Appointment Platform

An AI-powered full-stack doctor appointment booking platform built using MongoDB, Express, React, and Node.js. The system combines appointment management with intelligent healthcare assistance features to improve user experience and streamline doctor-patient interactions.

---

## 🚀 Key AI Features

- 🤖 AI-powered health assistance and smart interaction
- 🩺 Symptom-based intelligent guidance
- 📅 Smart doctor recommendation system
- 🔍 AI-assisted appointment suggestions
- 💬 Conversational chatbot support
- 📊 Intelligent patient data handling
- ⚡ Fast and responsive booking workflow

---

## ⚡ Run in 2 Steps (Windows)

### Step 1 — Install Node.js (One Time Setup)
Download and install Node.js LTS version:

https://nodejs.org

### Step 2 — Run the Application
Double-click:

`setup-and-run.bat`

The script automatically:

- ✅ Installs all required dependencies
- ✅ Configures environment variables
- ✅ Starts Backend server
- ✅ Starts Frontend application
- ✅ Starts Admin dashboard
- ✅ Opens browser automatically
